About UPISetu

UPISetu is a lightweight and efficient UPI-based payment management dashboard designed for small businesses, individuals, and organizations who want quick control over their transactions without heavy integration requirements.

This platform allows users to:

  • Generate custom UPI QR codes and payment links
  • Track and monitor recent and pending payments
  • Analyze earnings over time with simple visual charts

Database Table Structure

The following is the structure used in the connected Google Sheet for storing UPI transactions:

Column Name Description Example
Timestamp Date and time of transaction creation 3/27/2025 18:22:28

Name Payer's name (entered in form) John Doe

PaymentLink Generated UPI payment link upi://pay?pa=...
QR Code URL URL of generated QR code image https://api.qrserver.com/...
Status Current payment status Pending

Getting Started Guide

If you're setting up UPISetu for the first time, follow these steps:

  1. Copy the Google Sheet Template:
    • Create a new Google Sheet and set up the columns as shown in the table above (ensure exact column names).
  2. Deploy Google Apps Script:
    • Go to Google Apps Script and create a new project.
    • Connect it to your Google Sheet.
    • Paste your backend script (for creating QR codes, writing rows, reading data).
    • Deploy as a web app (with permissions set to "Anyone" or "Anyone with link").
  3. Paste Deployment ID into Settings Page:
    • Copy the web app URL after deployment (e.g., DEPLOYMENT_ID).
    • Open settings.html and paste this URL in the appropriate input field or script section.
    • This allows the frontend to communicate with your backend services.
  4. Host Your Frontend:
    • Upload the HTML files (index, create, about, etc.) to any static host like GitHub Pages, Vercel, or Netlify.
  5. Test Everything:
    • Submit a payment request from create.html.
    • Check that the entry is added to your sheet and displayed in transactions.html.
